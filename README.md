# HomoMinecraftLauncher

HomoMinecraftLauncher(HOML) is a Minecraft:JavaEdition Launcher for HarmonyOS NEXT.

HomoMinecraftLauncher(HOML)是一款适用于HarmonyOS NEXT的MinecraftJava版启动器。

> This software is currently in the early preview stage. It is not guaranteed to run perfectly, and problem feedback is not accepted for the time being.
> 
> 此软件目前处于早期预览时期，不保障可以完美运行，且暂不接受问题反馈。
